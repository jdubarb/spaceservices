# Space Services: Trade and Fuel

Addon staging folder for Space Services trade stops, fuel sales, and future commodity services.

Planned initial scope:

- Commercial service shuttle visits on stationary Space Services maps.
- Trader pawns that can be interacted with like caravan traders.
- Chemfuel sale requests through Space Services-owned fuel infrastructure.
- Short visit windows, safe-pad gating, danger gating, and deterministic traffic pacing.

Planned optional integrations:

- Dubs Bad Hygiene water sales.
- Vanilla Gravship Expanded astrofuel sales.
- Larger commercial pads and larger ship visuals.

This folder is intentionally staged before gameplay code exists.
